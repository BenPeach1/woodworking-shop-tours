# Woodworking Shop Tours

This is a simple HTML & Python application that displays the logos of several YouTube woodworking content creators. When one of the logos is clicked on, it will open and play the corresponding woodworker's most recent shop tour video.

## Input / Source
1) Image File
2) Video URL

## Output

When a woodworker's logo is clicked on, their corresponding shop tour youtube video will take focus and play on screen.
